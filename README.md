# solink_taskmagement_system
 GraphQL API for a task management application. 


clone repo: git clone https://github.com/zimmat/solink_taskmagement_system.git
cd solink_taskmagement_system

## Install dependences 
`npm run npm install`
## Run API
`npm run buil`
`npm run npm start`

## run migrations
`npm run typeorm-migrate`
